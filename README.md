# Friendo – Full-Stack Social Networking Platform

## Overview

Friendo is a full-stack social networking platform designed to connect users through an interactive and responsive web application. The platform enables users to engage with content, manage profiles, and experience seamless interactions through a modern user interface and backend architecture.

## Features

* User-friendly and responsive interface
* Profile management functionality
* Content sharing and interaction capabilities
* RESTful API integration
* Backend services for handling user operations
* Modern UI built with Tailwind CSS
* Scalable project structure with separate frontend and backend components

## Tech Stack

### Frontend

* React.js
* JavaScript
* Vite
* Tailwind CSS

### Backend

* Node.js

## Installation

### Clone the Repository
git clone https://github.com/vanshikasoni11/Friendo.git

### Navigate to the Project Directory
cd Friendo

### Install Dependencies
npm install

### Start the Application
npm run dev

### Start Backend Server
node server.js
